# 渡辺パイプ 顧客課題診断ツール

## 📋 プロジェクト概要

渡辺パイプ株式会社のライフライン事業部向け、顧客課題診断ツールのプロトタイプです。

営業マンが現場で顧客の課題をヒアリングし、即座に診断レポートを生成することで、以下を実現します：

- ✅ **渡辺パイプの営業効率向上**: 資材・調達・配送に関する具体的な提案を自動生成
- ✅ **顧客との関係深化**: 経営課題まで把握し、パートナーとしての価値を提供
- ✅ **BRANUへの自然なトスアップ**: 採用・集客課題を持つ顧客に対して、外部パートナーを紹介

---

## 🎯 主な機能

### 1. スタート画面
ツールの説明とメリットの提示

### 2. 基本情報入力
- 会社名の入力
- 業態選択（水工/電工/土木/設備/住設）

### 3. 課題ヒアリング（13項目のチェックリスト）

**渡辺パイプ提案領域（7項目）**
- 資材コスト高騰
- 資材調達の不安定さ
- 複数仕入れ先の管理負担
- 配送時間・費用の負担
- 資材ロス・余剰在庫
- 新商品・代替品情報の不足
- 施工方法の相談ニーズ

**経営課題領域（6項目）**
- 人材不足
- 若手採用難
- 高単価案件獲得難
- 新規顧客開拓難
- 見積作成の非効率
- 工程管理の属人化

### 4. 診断レポート
- 課題スコアの可視化
- 渡辺パイプの具体的な提案（優先表示）
- 外部パートナー紹介（BRANU株式会社等）
- 次のアクション選択

## 📝 使い方

### 営業現場での利用フロー

1. **顧客訪問時**
   - スマートフォンでツールにアクセス
   - 「診断を始める」をタップ

2. **基本情報の入力（30秒）**
   - 会社名を入力
   - 業態を選択

3. **課題のヒアリング（2分）**
   - 顧客と会話しながらチェック
   - 該当する項目をタップ

4. **診断レポートの確認（3分）**
   - その場で診断結果を表示
   - 顧客と一緒に画面を見ながら説明
   - 渡辺パイプの提案を前面に提示
   - 必要に応じて外部パートナーを紹介

5. **アクション選択**
   - メール送信
   - 提案書作成
   - 外部パートナー紹介
